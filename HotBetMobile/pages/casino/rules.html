GENERAL RULES

The slot is a single player casino game. This game starts by placing a bet and ends when the actual slot application is closed. Once opened, a single slot application, one can start a game session and every game session represents an independent new game which has no influence on any other game as each game session is unique.

The session is made up from different phases which are called "SPINS". The spins activates the reels to turn showing the winning symbols spinning fast and then stop to combine different combinations relative to the spin which has just taken place. Every symbol on the reel is set by the slot producer. At the end of each spin, it is possible to end the game and in this case, the game session is automatically ended.

The minimum and/or maximum bet varies according to the values set by the slot producer and vary from slot to slot. To bet, you need to utilise the coins and the value of these can be used in the "MENU BUTTONS" clicking on "COIN VALUE" ("MENU BUTTONS" and "COIN VALUE" are explained further on in this document) During a game session, one can fund his player account at any time but it is not possible to exceed the limits set by the game. After you choose the slot to play, choose the amount the total amount of coins you want to bet, using one of the buttons which are explained in the chapter "MENU BUTTONS" and click on "SPIN" to start the game. If you click on "AUTO", a menu will appear on the screen. This will be explained in the chapter named "MENU BUTTONS". Choosing "AUTO" allows a player to play automatically.

All prizes are relative to winning combinations starting from "LEFT" to "RIGHT" on the winning lines. On every line, the best winning combination is valid, in case of multiple winning combinations on a single line, only the highest combination is valid and is the one which pays the highest prize as explained in the "PAYTABLE".

PAYTABLE:

The "PAYTABLE" is the table showing the list of all symbols of the slot and their corresponding winning value. For every symbol of the slot you will find a list of every possible winning combination and the corresponding amount you can win with each combination.

With regards to the special symbols, you will find an additional description in the "PAYTABLE" which briefly explains what is described in more detail in the SYMBOLS chapter.

It also shows all winning combinations of all symbols and the amount won per combination.

In the slots one can also find small spaces or pop ups which contain mini extracts of the PAYTABLE" which show the player in brief, the winning combination for example : 4 x 100 symbol, which means that 4 of those symbols combined from left to right result in winning 100 coins.

These little popups come out casually in rotation.

SYMBOLS

  • Simple game symbols :

    The game symbols are listed in the "PAYTABLE", which is the table showing the payments where for every symbol, you will find a list of all winning combinations and the corresponding winning value.
    The simple game symbols are defined in this way because they have no special functions and only pay out the amount shown next to the winning combination which is shown in the "PAYTABLE".
    You can win with two or three similar symbols (depending on the type of slot) as long as they are adjacent (from left to right) and they appear on one of the lines activated by you.

  • Special game symbols:

    You can also find these symbols listed in the PAYTABLE and they are defined special because apart from paying the amount corresponding to the winning combination, these symbols have other functions like BONUS GAME or FREE GAMES

    • WILD
      The "WILD" symbol, if present, forms part of a winning combination, from two identical symbols to max of five identical symbols appear in an adjacent position (from left to right) and they are on one of the paying lines chosen by the player.
      The wild symbol can also replace all other symbols, thus completing a winning combination.
      The player can find the information on the symbol, and then check if it pays or not, on the "PAYTABLE."

    • SCATTER
      The "SCATTER" symbol, if present, forms part of a winning combination, from two identical symbols to max of five identical symbols that come out in any position they appear on the screen.
      The winning sequences with scatter symbols, if present, pay out the corresponding win and allows a player to access the "FREE GAMES". The free games will automatically start and during these spins, all general slot rules apply and in addition, if present, all winnings made ​​during the "FREE GAMES" could be multiplied by a factor that is shown in the "PAYTABLE". is indicated in the "PAYTABLE" next to each winning combination. Upon completion of the free games, the game will end with a summary of all winnings obtained and the game resumes to normal mode.

    • BONUS:
      The "BONUS" symbol, if present, forms part of a winning combination, from two identical symbols to max of five identical symbols appear in an adjacent position (from left to right) and they are on one of the paying lines chosen by the player.
      The winning sequences with bonus symbols, pay out the corresponding win and allows a player to access the "BONUS" game which is a mini-game.
      Upon completion of the bonus game, the game will end with a summary of all winnings obtained and the game resumes to normal mode and the player can continue to play.

MENU BUTTON HELP

On the menu button you can find areas with descriptions and some buttons.

  1. Descriptive Area: "BET"
  2. Button: "COINS PER LINE"
  3. Button: "LINES"
  4. Button: "SPIN"
  5. Button: "AUTO"
  6. Button: "BET MAX"
  7. Button: "COIN VALUE"
  8. Descriptive Area: "WIN"
  9. Occasional button "FREE GAMES"
Descriptive Area "BET":
The descriptive area "BET" shows the number of coins which have been bet by the player. This changes according to the values chosen by the player by using the "COINS PER LINE" and "LINES" button, the result of these will be shown in the descriptive area "BET"
Button "COINS PER LINE"
The "COINS PER LINE" button shows how many coins the player wants to be for every available betting line. The player, by clicking on this button, changes the value which starts from "0.01" and increases up to the maximum value set by the slot producer.
Button "LINES"
The "LINES" button shows how many lines the player wants to bet on. The higher the number of lines chosen, the better chance a player has to get a winning combination. When clicking on this button, the player changes the value of the lines which start from "1" and increases up to the maximum value set by the slot producer.
Button "SPIN":
The "SPIN" button is used by the player to start the game. By clicking on this button, the player will spin the slot wheels to play.
Button "AUTO":
The "AUTO" button can be clicked by the player who wants to set a number of continuous game plays automatically. Once the number of spins is chosen, the game is repeated for the number of times the player has chosen. The number of automatic game plays will show in the menu which appears as soon as a player choose to play in "AUTO" mode.
Button "BET MAX":
The "BET MAX" can 'be clicked by the player to set up and start immediately a play with the maximum number of lines and coins per line.
Button "COINS VALUE":
The "COINS VALUE" shows how many coins the player wants to bet. When clicking on this button, increases the value of the coins which will be bet in the game.
Descriptive Area "WIN":
The "WIN" descriptive area shows the amount won and is displayed when the player gets a winning combination in his current game.
Occasional "FREE GAMES" Button:
he occasional "FREE GAMES" button appears in place of the "SPIN" button when the player gets a winning combination which allow him to access the "FREE GAMES". This button automatically disappears from the menu button when all the free spins allowed have been used.
